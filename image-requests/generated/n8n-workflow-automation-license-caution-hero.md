## Concept

워크플로 노드, 라이선스 체크포인트, self-hosted 서버, 업무 승인 게이트가 함께 보이는 고급 업무 자동화 검토 이미지.

## Must Include

- workflow nodes
- license checkpoint
- self-hosted server
- approval gate
- business automation map

## Must Avoid

- official n8n logo
- third-party SaaS logos
- real credentials
- claiming open-source without caution

## Safety

- Do not call external SaaS APIs or send messages from article examples.
- Do not expose customer credentials, API keys, or workflow secrets.
- Treat production workflow automation as a separate approval-gated implementation.
