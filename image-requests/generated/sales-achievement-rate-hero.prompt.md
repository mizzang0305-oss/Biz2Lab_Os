# Biz2Lab Image Prompt Package

## Provider Prompt (Korean)
매출 달성률 해석을 실제 업무 흐름 중심으로 설명하는 hero 인라인 이미지. 입력값, 상태값, 담당자 확인, 다음 조치가 보이되 실제 회사명, 고객명, 로고, 개인정보는 넣지 않는다. 작은 글자를 많이 넣지 말고 표, 카드, 화살표, 상태 칩을 사용한다.

## Provider Prompt (English)
Create a safe Biz2Lab inline hero image for 매출 달성률 해석. Use workflow cards, status chips, and clear operational structure. Avoid logos, people, private data, fake screenshots, and tiny unreadable text.

## Negative Prompt
실제 로고, 실명, 고객 정보, 계좌 정보, 카드 정보, 사람 얼굴, 외부 URL, 워터마크, 읽기 어려운 작은 글자

## Alt Text
매출 달성률 해석을 설명하는 대표 이미지

## Caption
매출 달성률 해석에서 입력값, 담당자 확인, 다음 조치를 한눈에 보도록 정리한 대표 이미지입니다.

## Output Paths
- rawPath: assets/images/raw/sales-achievement-rate-hero.svg
- optimizedPath: public/images/posts/sales-achievement-rate-hero.webp

## Validation
- localOnly: true
- outputMode: prompt-only
- no external image URL
- no private data
