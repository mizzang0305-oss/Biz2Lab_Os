## Must Include

- database grid

## Must Avoid

- official Baserow logo
- Airtable logo
- real customer data
- dense table text

## Safety

- Do not include real customer data in examples.
- Do not claim a database is privacy-safe without deployment and access-control verification.
- Backups and permission roles must be treated as production gates.
