## Concept

Lightweight backend, auth module, database module, file storage, backup and scaling gate가 있는 SaaS MVP 운영 이미지.

## Must Include

- lightweight backend
- auth module
- database module
- file storage
- backup and scaling gate

## Must Avoid

- official PocketBase logo
- real user accounts
- payment data
- production launch claim
- unbounded scaling promise

## Safety

- Do not connect production customer data, payment data, or private business systems in examples.
- Use sample data and owner-approved test environments before any production integration.
- Treat authentication, permissions, audit logs, backups, and rollback as publication-time gates.
