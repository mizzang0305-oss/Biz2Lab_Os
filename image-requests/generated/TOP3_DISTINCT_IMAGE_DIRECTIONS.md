# TOP3 Distinct Image Directions

Status: prompt_repair_for_visual_uniqueness
Date: 2026-06-16

Use these directions before generating or accepting any TOP3 premium hero image. The three visuals must not share the same dashboard/card composition. Generate one image at a time, audit uniqueness after each accepted raw file, and do not apply article frontmatter until all three raw images are visually distinct.

## Shared Uniqueness Rules

- Do not generate all three in one batch.
- Do not reuse the same source file, composition, raw path, optimized path, or visual layout across slugs.
- Do not use deterministic SVG fallback, placeholder cards, or repeated template diagrams as final premium output.
- Same dimensions are expected and are not a failure; same hash, same source, or near-identical visual layout is a failure.
- If a candidate is duplicate or near-duplicate, move it to `assets/images/rejected/[slug]-duplicate-[timestamp].png`.
