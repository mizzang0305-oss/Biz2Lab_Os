# AdSense 엔터테인먼트 전면 개편 검증 보고서

## 결론

- 공개 글 22개: 영화 선택 8개, 결말·주제 해석 8개, OTT 생활 6개
- 보류 글 53개: 기존 업무 자동화·매출 관리 글은 `draft: true`, `noindex: true`로 공개 대상에서 제외
- 외부 배포: 하지 않음
- AdSense 재검토 요청: 하지 않음
- 가짜 감상, 가짜 조회수, 검색 순위를 만들지 않음

글 수는 승인 공식이 아니라 사이트가 한 주제를 충분히 설명하는지 확인하기 위한 운영 범위다. Google이 공개한 정책에는 최소 글 수나 최소 글자 수가 승인 조건으로 제시되어 있지 않으므로, 22개라는 숫자를 승인 보장으로 표현하지 않는다.

## 사용자 가치 전환

기존 첫 화면의 미수금·매출·자동화 중심 글은 현재 실제 조회가 없고 일반 독자가 머무를 이유가 약했다. 공개 목적을 다음 세 가지로 다시 좁혔다.

1. 오늘의 기분, 남은 시간, 동행에 맞는 영화 선택
2. 결말과 주요 장면을 다시 읽는 영화 해석
3. Netflix 설정과 OTT 구독을 덜 헤매게 하는 생활 도움말

모든 공개 글은 서로 다른 편집자 메모, 독자 대상 3개 이상, FAQ 3개 이상, H2 5개 이상, 관련 글 연결과 스포일러 범위를 갖는다. 공개 본문은 최소 1,200자이며, 반복 문단과 과도한 템플릿 유사도를 자동 검사한다.

## 이미지·저작권 처리

《인사이드 아웃 2》 글에 공식 키아트 1장과 장면 스틸 2장을 사용했다. 세 파일은 Disney UK Press가 각 공식 갤러리에서 `Editorial use only`로 표시한 보도 자료다.

- 키아트 출처: https://press.disney.co.uk/gallery/inside-out-2-logo-and-key-art
- 장면 스틸 출처: https://press.disney.co.uk/gallery/inside-out-2
- 권리자: Disney
- 출처·사용 근거 확인일: 2026-07-17
- 저장 방식: 원본 출처 URL을 권리 인벤토리에 남기고, 화면 슬롯에 맞춘 WebP 파생본만 로컬에 저장
- 표시 방식: 이미지 아래에 저작권 고지, 권리자, 확인일과 공식 출처 링크 노출

KOFIC 자료는 복제·재배포 제한이 있고, TMDB는 광고 기반 추천 사이트에 별도 상업 계약을 요구하므로 이미지 원본으로 사용하지 않았다. 라이선스가 불명확한 다른 작품의 포스터와 스틸은 넣지 않았으며, 승인 가능한 공식 프레스 자료를 찾은 경우에만 같은 인벤토리 절차로 추가한다.

## 정직성·AdSense 관점

- 직접 보지 않은 작품의 체험담, 시사회 참석, 업계 경력을 만들지 않는다.
- 인기 순위와 개인 추천을 구분하고, 실제 분석 도구가 연결되기 전에는 조회수·검색량·순위를 표시하지 않는다.
- AI는 검색어 후보, 구조화와 누락 점검에만 사용하며 최종 문장·해석·공식 출처·공개 여부를 다시 검토한다.
- 22개 글의 게시일은 실제 개편일인 2026-07-17로 동일하게 기록하고 가짜 과거 날짜를 만들지 않는다.
- AdSense 승인을 보장하지 않는다. 이번 변경은 저가치 콘텐츠 사유를 줄이기 위한 품질 개선이며 최종 판단은 Google 검토에 달려 있다.

## 배포 경계

이 보고서와 구현은 별도 로컬 브랜치에서만 준비했다. 라이브 사이트, `master`, Vercel 배포, Search Console 제출, AdSense 재검토 요청은 변경하지 않았다.
