# AdSense Approval Checklist

- Korean-only public pages.
- About, contact, privacy, terms present.
- No public `/en`, `/ja`, `/admin`, `/ai`, `/commerce`, `/affiliate`, `/tools`, `/lotto`.
- Original posts with useful internal links.
- Local images with alt text.
- Sitemap and robots configured.
- No affiliate links before approval.
- No login, comments, chatbot, or public admin.
- Validate with `npm run validate:posts`, `npm run validate:seo`, `npm run check:links`, and `npm run build`.
