# Biz2Lab Platform Foundation

Current public product:

- Korean-only AdSense approval blog.
- File-based Markdown posts.
- Static category hubs.
- Optional Supabase capture for contact and newsletter only.

Future internal engines:

- Content Factory: Obsidian, Markdown, SEO rules, image rules.
- Research Engine: source ledger and crawler policy.
- Asset Engine: local image rules and future shared models.
- Memory Engine: future pgvector/RAG.
- Growth Engine: Search Console, click events, CTA performance.

Public routes must remain limited to the approved Korean MVP route set until AdSense approval.
