## Top-Level Shape

```json
{
  "version": 1,
  "project": "biz2lab",
  "language": "ko",
  "policy": "local-generated-only",
  "briefs": []
}
```

## Brief Entry Fields

Required fields:

- `id`: Stable unique asset brief id.
- `postSlug`: Post slug, or `hub-...` for future hub summary visuals.
- `category`: One of the Biz2Lab Korean public categories.
- `usage`: `hero`, `inline`, or `hub-summary`.
- `targetPath`: Planned raw local image path under `assets/images/raw/`.
- `optimizedPath`: Planned public WebP path under `public/images/posts/`.
- `altKo`: Korean alt text that can replace the image meaning.
- `captionKo`: Korean caption for article or manifest use.
- `style`: Visual style constraints.
- `promptKo`: Korean local generation prompt.

## Validation Rules

- `targetPath` must start with `assets/images/raw/`.
- `optimizedPath` must start with `public/images/posts/`.
- `optimizedPath` must end with `.webp`.
- `altKo`, `captionKo`, and `promptKo` must contain Korean text.
- No brief may request real logos, real customer data, product images, Amazon images, or private documents.
- Generated prompt packages must include `negativePromptKo`, `categoryStyle`, and `visualDifferentiationHint`.
- Missing raw files are warnings until image generation is explicitly run.
- Missing optimized files are warnings until `npm run optimize-images` is explicitly run.
- A missing local-only generation command means every brief remains pending; do not mark a brief failed unless a real local generation attempt was made.
- A brief is ready for manifest activation only after its raw file exists and the optimized WebP exists under `public/images/posts/`.

## Naming Rules

Hero raw files:

```text
assets/images/raw/{post-slug}-hero.png
```

Hero optimized files:

```text
public/images/posts/{post-slug}-1200.webp
public/images/posts/{post-slug}-800.webp
public/images/posts/{post-slug}-400.webp
```

Inline raw files:

```text
assets/images/raw/{post-slug}-inline-{topic}.png
```

Inline optimized files:

```text
public/images/posts/{post-slug}-inline-{topic}-1200.webp
```

Hub summary raw files:

```text
assets/images/raw/hub-{category}-summary.png
```
