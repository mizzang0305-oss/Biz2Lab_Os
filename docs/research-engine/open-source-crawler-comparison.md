# Open Source Crawler Comparison

- Firecrawl: API-oriented extraction; useful for structured research when policy allows.
- Crawl4AI: AI-oriented crawling pipeline; useful for controlled experiments.
- Crawlee: robust JavaScript crawler framework; useful for queue and browser crawling.
- Trafilatura: strong article extraction for text research.
- Mozilla Readability: page cleanup for article-like pages.
- Playwright-based crawlers: powerful but heavier; use only when simple fetch extraction is not enough.

Selection criteria: robots respect, source attribution, rate limits, repeatability, and legal/ethical fit.
